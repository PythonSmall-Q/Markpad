# Markpad 使用指南

## 功能快速上手

### 1. 创建和管理文档

#### 新建文档
- 点击顶部工具栏的 **"新建"** 按钮
- 或使用快捷键 `Ctrl+N` (Windows/Linux) / `Cmd+N` (Mac)
- 新文档将以 "未命名文档" 显示在标签栏

#### 打开文件
- 点击 **"打开"** 按钮选择文件
- 或使用快捷键 `Ctrl+O`
- 支持的格式：`.md`、`.markdown`、`.txt`
- 已打开的文件会显示在侧边栏的"已打开"列表中

#### 保存文档
- 点击 **"保存"** 按钮
- 或使用快捷键 `Ctrl+S`
- 首次保存会弹出"另存为"对话框
- 文档标题旁的蓝点 ● 表示有未保存的修改

#### 另存为
- 点击 **"另存为"** 按钮
- 可以将当前文档保存到新位置或以新名称保存

#### 关闭文档
- 点击标签上的 ✕ 按钮
- 或使用快捷键 `Ctrl+W`
- 如果有未保存的修改，会提示确认

### 2. Markdown 编辑

#### 基础语法

**标题**
```markdown
# 一级标题
## 二级标题
### 三级标题
```

**文本样式**
```markdown
**粗体文本**
*斜体文本*
~~删除线~~
`行内代码`
```

**列表**
```markdown
- 无序列表项 1
- 无序列表项 2

1. 有序列表项 1
2. 有序列表项 2
```

**链接和图片**
```markdown
[链接文本](https://example.com)
![图片描述](图片路径)
```

**代码块**
````markdown
```javascript
function hello() {
  console.log('Hello, World!')
}
```
````

**引用**
```markdown
> 这是一段引用文本
> 可以多行
```

**表格**
```markdown
| 列1 | 列2 | 列3 |
| --- | --- | --- |
| 内容1 | 内容2 | 内容3 |
```

**分隔线**
```markdown
---
```

#### 使用工具栏

顶部的编辑器工具栏提供了快捷按钮：
- **B**: 加粗选中文本
- **I**: 倾斜选中文本
- **S**: 删除线
- **标题**: 下拉选择标题级别
- 列表按钮: 创建无序/有序列表
- 引用按钮: 创建引用块
- 代码按钮: 插入行内代码或代码块
- 链接按钮: 插入链接
- 图片按钮: 插入图片
- 表格按钮: 插入表格
- 分隔线按钮: 插入水平分隔线
- 预览按钮: 切换预览显示/隐藏

### 3. 插入文件

#### 插入图片
1. 点击工具栏的图片按钮
2. 选择本地图片文件
3. 图片会以 Markdown 语法插入到编辑器中
4. 支持格式：JPG、PNG、GIF、SVG、WebP

#### 插入视频/音频
- 使用标准的 HTML 标签：
```html
<video src="视频路径" controls></video>
<audio src="音频路径" controls></audio>
```

### 4. 导出文档

点击顶部的 **"导出"** 下拉菜单，选择导出格式：

#### Markdown 导出
- 导出为 `.md` 文件
- 保留原始 Markdown 格式
- 适合在其他 Markdown 编辑器中使用

#### HTML 导出
- 导出为 `.html` 文件
- 包含完整的样式
- 可以在浏览器中直接打开
- 适合发布到网页

#### PDF 导出
- 导出为 `.pdf` 文件
- 保留格式和样式
- 适合打印和分享

#### 纯文本导出
- 导出为 `.txt` 文件
- 去除所有 Markdown 格式
- 仅保留纯文本内容

### 5. 界面和主题

#### 切换主题
- 点击右上角的月亮/太阳图标
- 在亮色和暗色主题之间切换
- 主题设置会自动保存

#### 显示/隐藏预览
- 点击工具栏的眼睛图标
- 切换预览面板的显示状态
- 隐藏预览可以获得更大的编辑空间

#### 侧边栏
- 显示最近打开的文件
- 显示当前已打开的所有文档
- 点击文件可以快速切换
- 点击 ✕ 可以关闭文档

### 6. 快捷键

| 功能 | Windows/Linux | Mac |
| --- | --- | --- |
| 新建文档 | Ctrl+N | Cmd+N |
| 打开文件 | Ctrl+O | Cmd+O |
| 保存文件 | Ctrl+S | Cmd+S |
| 关闭文档 | Ctrl+W | Cmd+W |
| 搜索 | Ctrl+F | Cmd+F |

### 7. 自动保存

- 应用会每 60 秒自动保存已打开的文件
- 只有已经保存过的文件（有文件路径）才会自动保存
- 新建的未命名文档需要手动保存一次
- 自动保存不会弹出提示

### 8. 最近文件

- 最近打开的文件会显示在欢迎页和侧边栏
- 最多保留 10 个最近文件
- 如果文件被删除或移动，点击后会提示错误并自动从列表中移除

## 高级技巧

### 1. 多文档工作流

- 同时打开多个文档，在标签之间快速切换
- 参考一个文档的内容编辑另一个文档
- 使用侧边栏快速定位和切换文档

### 2. 图片管理

- 建议将图片保存在项目文件夹内
- 使用相对路径引用图片
- 这样整个项目可以方便地移动和分享

### 3. 代码块

- 指定语言可以获得语法高亮
- 支持常见编程语言：JavaScript、Python、Java、C++、HTML、CSS 等

### 4. 导出策略

- **日常编辑**: 使用 Markdown 格式保存
- **分享给非技术人员**: 导出为 PDF
- **发布到网站**: 导出为 HTML
- **纯文本需求 **: 导出为 TXT

## 常见问题

### Q: 如何插入本地图片？
A: 点击工具栏的图片按钮，选择本地图片文件，系统会自动插入 Markdown 图片语法。

### Q: 预览和编辑不同步？
A: 编辑内容后预览会自动更新，如果没有更新，尝试重新打开文档。

### Q: 自动保存失败？
A: 确保文件没有被其他程序占用，检查文件权限是否正确。

### Q: 导出的 PDF 样式不对？
A: PDF 导出使用浏览器的打印功能，样式与预览面板一致。

### Q: 如何备份我的文档？
A: Markdown 文件就是纯文本文件，可以直接复制到其他位置备份，或使用云同步服务。

### Q: 支持云同步吗？
A: 当前版本不内置云同步，但你可以将文档保存在 OneDrive、Dropbox 等云盘文件夹中实现同步。
